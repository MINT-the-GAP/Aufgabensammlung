# Rechenmauer der Additon

**Bestimme** die fehlenden Felder der Rechenmauer. Hierbei werden zwei benachbarte Felder miteinander addiert und die Summe in dem Feld darüber eingetragen.

``` ascii
                  +-----------------+

                  | " [[  14   ]] " |

         +--------+--------+--------+--------+
         |                 |                 |

+--------+--------+--------+--------+--------+--------+
|                 |                 |                 |
|      " 4 "      |      " 2 "      |      " 6 "      |

```
